Overview 
These tasks are designed based on Week 1 interview insights and focus on the most critical onboarding pain points: 
- Lack of clear starting point 
- Difficulty finding information 
- Unclear next steps 
- Not knowing who to ask for help 
 
Each task includes a realistic scenario and measurable success criteria. 
 

Task 1 — Start Your First Day (Onboarding Entry) 
 
Scenario 
You have just joined a new company. You log into the platform for the first time. 
 
Task 
Show how you would start your first day. 
 
What We Are Testing 
- Clarity of the onboarding entry point 
- Ability to understand where to begin without guidance 
 
Success Criteria 
- User identifies a clear starting point within **10 seconds** 
- User does not ask clarifying questions 
- User expresses confidence in what to do next 
 
  Metrics 
- Time to first click 
- Number of hesitations 
- Confidence rating (1–5) 
 
Task 2 — Understand What to Do Next 
 
Scenario 
You have completed your first onboarding step and want to continue your work. 
 
Task 
Find what you should do next. 
 
What We Are Testing 
- Clarity of task progression 
(Optional) Task 5 — Evaluate AI Response Trust 
 
Scenario 
You ask the system a question and receive an AI-generated response. 
 
Task 
Review the response and decide if you trust it. 
 
What We Are Testing 
- Trust in AI-generated answers 
- Importance of sources and explanations 
 
Success Criteria 
- User checks source or explanation 
- User expresses trust when sources are visible 
- User identifies when AI should not be trusted 
 
Metrics 
- Trust rating (1–5) 
- Whether sources are used 
- User feedback on clarity
